## Filosofía

El jugador comienza sin nada.

Cada herramienta representa un avance.

El mundo debe sentirse vivo.

La progresión debe sentirse constante.

Nunca debe existir un momento donde el jugador piense:

"Ya no tengo nada para hacer."
