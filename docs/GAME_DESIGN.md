# GAME DESIGN DOCUMENT

## Nombre

Wild Origin

## Género

Supervivencia

Sandbox

Construcción

Exploración

## Cámara

Vista superior (Top Down)

Zoom controlable con la rueda del mouse (interpolado, no instantáneo)

## Mundo

Infinito, generado por chunks a medida que el jugador se mueve

Cada chunk se genera de forma determinística a partir de una semilla, así que descargar y volver a cargar una zona no cambia su contenido

Biomas suaves definidos por ruido 2D continuo (zonas más boscosas, más rocosas, etc.), sin costuras entre chunks

## Estilo gráfico

Pixel Art

Colores naturales

Bosques

Luz dinámica

## Inspiraciones

Minecraft

Don't Starve

Terraria

Valheim

## Objetivo del jugador

Sobrevivir.

Explorar.

Construir.

Evolucionar.

## Filosofía

El jugador comienza sin nada.

Cada herramienta representa un avance.

El mundo debe sentirse vivo.

La progresión debe sentirse constante.

Nunca debe existir un momento donde el jugador piense:

"Ya no tengo nada para hacer."

## Prioridades

1. Diversión

2. Jugabilidad

3. Rendimiento

4. Gráficos
